# About

"SquidGame" is a popular Netflix series that has been trending in recent days. This is a (not complete) recreation of the series.

### Features

- Arena Setup GUI.
- Multi-game arena sequence.
- High performance.
- Multi-arena Support.
- Open Source.
- PlaceholderAPI Support.
- Translation system.
- Admin arena controls.
- Automatic config/message default merging on updates.
