# SquidGame

Minecraft Minigame inspired by the famous and trending Netflix series "Squid Game"

### Features

- [x] 1.8 Support  
- [ ] Better configuration  
- [ ] Bungee-arena Support  
- [ ] Developer API  
- [ ] Dynamic Translation System  
- [ ] Leaderboards  
- [ ] Min and max players per arena  
- [ ] Player statistics  
- [ ] Teaming Mode
